## Project Name
Food Ordering Service

## Description
This project is an implementation of a service that provides the ability to order food online. Here, customers have the opportunity to purchase various products, as well as indicate their type, quantity needed, delivery and payment method. These records come from various manufacturing companies. Using a database, this system allows you to display all the available products in the catalog, so that users can select products and place orders.

## Installation
Here is the instruction on how to install this application locally:
1) Download this application from the official online-shop.
2) Build it from the terminal using make.
3) Run application in the terminal or simply use the file bin/food_service.

## Usage
Here are some guidelines for working with the application:
1) Register in the system.
2) Log in.
3) Using the database, select any category or property to find the right product from the manufacturer.
4) Enter the necessary parameters for the search, if you need.
5) Enter your details to purchase the necessary goods so that you can be contacted and serviced later.
